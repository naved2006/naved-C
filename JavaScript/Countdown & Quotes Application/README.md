# 🚀 Event Countdown Project

## 🔹 Project Overview

This project combines three JavaScript mini-projects into one application:

* ⏳ Event Countdown Timer
* 💬 Motivational Quotes Slider
* 🎉 Welcome Modal Popup

The project displays a live countdown to a future event, shows motivational quotes automatically, and displays a welcome popup after page load.

---

## 🔹 Features

### ⏳ Countdown Timer

* Live countdown to a future event
* Displays Days, Hours, Minutes & Seconds
* Start button
* Pause button
* Updates every second using `setInterval()`
* Shows "🎉 Time's up!" when countdown ends

### 💬 Motivational Quotes Slider

* Multiple motivational quotes stored in an array
* Auto quote change every 4 seconds
* Previous button
* Next button
* Dynamic content update using JavaScript

### 🎉 Modal Popup

* Automatically appears after 5 seconds
* Welcome message display
* Close button functionality
* Close when clicking outside modal

---

## 🔹 How This Project is Made

* HTML se project ka basic structure banaya gaya hai
* CSS ka use karke modern responsive UI design ki gayi hai
* Countdown section, Quotes section aur Modal section create kiye gaye hain

### JavaScript ka use karke:

#### Countdown Timer

* Future event date set ki gayi hai
* Current time aur event time ka difference calculate kiya gaya hai
* `setInterval()` se timer har second update hota hai
* `clearInterval()` se timer stop kiya gaya hai

#### Quotes Slider

* Quotes array create ki gayi hai
* Current quote display ki jati hai
* Next aur Previous buttons se quotes change hote hain
* Auto quote slider ke liye `setInterval()` use kiya gaya hai

#### Modal Popup

* `setTimeout()` se modal 5 seconds baad show hota hai
* Close button se modal hide hota hai
* Outside click par bhi modal close hota hai
