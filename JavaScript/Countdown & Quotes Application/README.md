# 🚀 Event Countdown Project

## 🔹 Project Overview

This project combines three JavaScript mini-projects into one application:

* ⏳ Event Countdown Timer
* 💬 Motivational Quotes Slider
* 🎉 Welcome Modal Popup

The project displays a live countdown to a future event, shows motivational quotes automatically, and displays a welcome popup after page load.

---

## 🔹 Features

### ⏳ Countdown Timer

* Live countdown to a future event
* Displays Days, Hours, Minutes & Seconds
* Start button
* Pause button
* Updates every second using `setInterval()`
* Shows **"🎉 Time's up!"** when countdown ends

### 💬 Motivational Quotes Slider

* Multiple motivational quotes stored in an array
* Auto quote change every 4 seconds
* Previous button
* Next button
* Dynamic content update using JavaScript

### 🎉 Modal Popup

* Automatically appears after 5 seconds
* Welcome message display
* Close button functionality
* Close when clicking outside modal

---

## 🔹 How This Project is Made

* HTML is used to create the project structure.
* CSS is used to design a modern and responsive user interface.
* Separate sections are created for Countdown Timer, Quotes Slider, and Modal Popup.

### JavaScript Implementation

#### Countdown Timer

* Future event date is set using the Date object.
* Current time and event time difference is calculated.
* `setInterval()` updates the countdown every second.
* `clearInterval()` stops the timer when needed.

#### Quotes Slider

* Quotes are stored in an array.
* Current quote is displayed dynamically.
* Previous and Next buttons allow manual navigation.
* Automatic quote changing is handled using `setInterval()`.

#### Modal Popup

* `setTimeout()` displays the modal after 5 seconds.
* Close button hides the modal.
* Clicking outside the modal also closes it.
