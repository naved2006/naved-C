# 🛍️ E-Commerce Furniture Website

A modern and responsive Furniture E-Commerce website built using **HTML5, CSS3, and JavaScript**. This project includes multiple pages such as Home, About, Contact, Sign In, Product Categories, Cart, and more.

---

## 🚀 Features

- 🏠 Responsive Home Page
- 🛏️ Bedroom Collection
- 🛋️ Living Room Collection
- 🍽️ Dining Collection
- 🛒 Shopping Cart
- 📂 Cart Drawer
- 👤 Sign In Page
- 📞 Contact Page
- ℹ️ About Page
- 📱 Mobile Friendly Design
- 🎨 Modern UI
- ⚡ Fast Loading

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
