# 🛍️ Product Manager - JavaScript CRUD Project

A simple and responsive **Product Manager** application built using **HTML, CSS, and JavaScript**. This project demonstrates complete **CRUD (Create, Read, Update, Delete)** operations with **Local Storage**, **Search**, **Filter**, and **Sorting** functionality.

---

## 📌 Features

### ✅ CRUD Operations
- Add Product
- Edit Product
- Delete Product
- Update Product

### ✅ Product Details
- Product Title
- Product Price
- Product Image URL
- Product Category

### ✅ Search & Filter
- Search products by title
- Filter by category
- Reset filters

### ✅ Sorting
- Price Low → High
- Price High → Low

### ✅ Local Storage
- Products are saved in browser Local Storage.
- Data remains available even after refreshing the page.

### ✅ Responsive Design
- Desktop
- Tablet
- Mobile Friendly

---

# ⚙️ Functionalities

## ➕ Add Product
Users can add a new product by entering:
- Title
- Price
- Image URL
- Category

Validation is applied before adding.

---

## ✏️ Edit Product
Click the **Edit** button to update product details.

---

## 🗑️ Delete Product
Click **Delete** to permanently remove a product.

---

## 🔍 Search
Instantly search products by title.

---

## 📂 Category Filter
Display products based on selected category.

---

## 💰 Price Sorting
Sort products:

- Low to High
- High to Low

---

## 💾 Local Storage
All products are automatically stored in Local Storage.

```
localStorage
```

No database required.
